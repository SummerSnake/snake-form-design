# 项目

snake-form-design

# 说明

表单设计器
React + Typescript + webpack5.x + ant design + dva

# 注意事项

# 安装

```

npm install snake-form-design --save

yarn add snake-form-design

```

# 示例

```

import SnakeFormDesign from 'snake-form-design';
import 'snake-form-design/dist/index.css';

const App = () => (
  <>
    <SnakeFormDesign />
  </>
);

```
